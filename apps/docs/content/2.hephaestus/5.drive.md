# Drive 

You can also manipulate your files using drive composable

```js 
:: setup
import { drive } from "app:drive"
import { ref } from "vue"

const items = ref([])

async function setItems() {
    const entries = await drive.list('/')
    
    items.value = entries.map(e => e.name)
}

setItems()

::

# Drive

## Root entries

<pre>{{ items }}</pre>
```
